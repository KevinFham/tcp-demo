# Order-Preserving, Lossless TCP via UDP sockets Demo 

For academic projects and for fun

## Python Instructions

Simply run the `python/main.py` script with a file path to a txt file

```bash
python3 python/main.py <path-to-txt-file>

```

For example:

```bash
python3 python/main.py tests/test-normal.txt 
```

## C Instructions

Compile `c/main.c`

```bash
gcc c/main.c -o main
```

Run the program with a file path to a txt file. For example:

```bash
./main tests/test-normal.txt
```
